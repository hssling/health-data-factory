# Security Policy

## Supported versions
This project currently supports the latest `main` branch.

## Reporting vulnerabilities
Report issues privately to the maintainers before public disclosure. Include:
- impact summary
- reproduction steps
- affected versions/paths

## Data safety controls
- PII heuristic failure gate in build pipeline.
- Explicit compliance checks for scraping behavior.
- Immutable manifests with file hashes and provenance.
